Universal Translator for AI

Model Context Protocol(MCP)은 AI 모델이 다양한 데이터 소스 및 도구와 원활하게 통신할 수 있도록 돕는 개방형 표준입니다. 마치 USB 포트가 하드웨어 연결을 표준화한 것처럼, MCP는 AI 애플리케이션의 데이터 통합 방식을 표준화합니다.

Quickstart for Python

TS/JS Integration

npm install @modelcontextprotocol/sdk
